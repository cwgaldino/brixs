brixs.Peaks

class brixs.Peaks(*args, **kwargs)

Creates a Peaks object.

How to create a peaks object:

p = br.Peaks()

p = br.Peaks(filepath=<filepath>) p = br.Peaks(<filepath>)

The filepath must point to a file with a JSON string.
