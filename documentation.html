Core

BRIXS is divided in four major classes:

brixs.Spectrum(*args, **kwargs)

Returns a Spectrum object.

brixs.Spectra(*args, **kwargs)

Returns a spectra object.

brixs.Image(*args, **kwargs)

Returns a Image object.

brixs.PhotonEvents(*args, **kwargs)

Returns a Photon events object.

and one secondary class:

brixs.Peaks(*args, **kwargs)

Creates a Peaks object.
